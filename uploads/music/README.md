# 音乐文件夹

此文件夹用于存放网站背景音乐文件。

## 目录结构
```
music/
├── 2025/
│   └── 12/
│       └── xxx.mp3      # 音乐文件按年月存放
└── covers/
    └── 2025/
        └── 12/
            └── xxx.jpg  # 封面图片按年月存放
```

## 支持格式
- 音乐：MP3、WAV、OGG、M4A
- 封面：JPG、PNG、WEBP

## 使用说明
1. 在管理后台的"音乐管理"中点击"添加音乐"
2. 点击"选择音乐文件"上传音乐
3. 可选上传封面图片
4. 填写歌曲名称和歌手信息
5. 保存后即可在前台播放器中播放
